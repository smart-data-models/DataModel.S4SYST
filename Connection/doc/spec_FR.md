<!-- 20-Description -->  
Description globale : **La classe des connexions entre les systèmes. Cette classe qualifie la propriété s4syst:connectedTo. Une connexion décrit les interactions potentielles entre les systèmes. Deux systèmes connectés sont reliés par une connexion. Une connexion peut relier plus de deux systèmes en même temps**.  
version : 0.0.1
