Description globale : **La classe des systèmes, c'est-à-dire des systèmes virtuellement isolés de l'environnement, dont le comportement et les interactions avec l'environnement sont modélisés. Les systèmes peuvent être connectés à d'autres systèmes. Les systèmes connectés interagissent d'une certaine manière. Les systèmes peuvent également comporter des sous-systèmes. Les propriétés des sous-systèmes contribuent d'une manière ou d'une autre aux propriétés du supersystème.    
version : 0.0.1
