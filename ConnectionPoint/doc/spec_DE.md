<!-- 20-Description -->    
Globale Beschreibung: **Die Klasse der Verbindungspunkte von Systemen, an denen sie mit anderen Systemen verbunden werden können. Diese Klasse qualifiziert die Eigenschaften s4syst:connectsSystem und s4syst:connectedThrough. Ein Anschlusspunkt gehört zu genau einem System. Jedes System, das über einen Anschluss verbunden ist, ist an einem seiner Anschlusspunkte mit dem Anschluss verbunden. Das System eines Verbindungspunktes, das über eine Verbindung verbunden ist, ist selbst über die Verbindung verbunden.**    
Version: 0.0.1
